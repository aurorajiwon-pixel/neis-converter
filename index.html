구글 시트로 옮겨 점수 조회 시스템 만들기

  1. 다운로드한 파일을 구글 드라이브에 업로드

    드라이브에서 우클릭 → 'Google Sheets로 열기' → '파일 → Google Sheets로 저장'으로 정식 변환하세요.

  2. 구글 폼 만들기

    질문 3개로 단순하게: 학번 (5자리 숫자), 이름, 비밀번호. 폼 응답을 위에서 만든 시트와 연결하세요.

  3. 폼 응답 시트 이름을 '비밀번호'로 변경

    폼 연결하면 새 시트가 자동 생성됩니다. 이름을 정확히 비밀번호로 바꿔주세요. 기본 '비밀번호' 시트는 삭제.

  4. Apps Script로 조회 웹앱 배포

    확장 프로그램 → Apps Script → 김지원 선생님이 공유한 Code.gs, index.html 코드 붙여넣기 → 배포 → 웹앱.
